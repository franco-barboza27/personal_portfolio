## Steps for use
1. Open VS code, or a codespace (DISCLAIMER: May break if using a codespace)
2. Go to the main.py file in this repo.
3. press the RUN button.
4. Have fun!

## List of KEY features
- GUI
    - This program will entirely be GUI based, using TKinter
        - in other words it will have visual aspects outside of the terminal.

- Morse Code 
    - Translates between morse code and alphabetical letters. :smiley:

- Book Library
    - A library that allows you to add, view, and remove books
        - DISCLAIMER: Does not save between runs :sob:

- Password Generator :1234:
    - Generates a strong password
        - Password strength is decided by:
            - Length
            - Special characters
            - Lower case letter
            - Capitals
            - Numbers

- Finance Tracker
    - Allows you to keep track of:
        - income, budget, savings, and more!
    - Don't worry about your data being harvested, it doesn't save :zany_face:
